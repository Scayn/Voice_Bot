# Privacy Policy

Last updated: April 2026

## What we collect
This bot stores the following per server:
- Voice channel IDs configured as dynamic channels
- Channel name preferences and name pools

## What we do NOT collect
- Usernames or user IDs
- Message content
- Any personal information

## Data removal
To remove your server's data, use /vb remove or kick the bot from your server.
All associated configuration is deleted automatically.

## Contact
If you have any questions, please open an issue on our GitHub repository.
